# NumPy — Applied Real Data Analysis

Hands-on NumPy practice using real datasets. No pandas, no matplotlib — just arrays, logic, and NumPy's full toolkit.

Each notebook is a standalone analysis with its own dataset, questions, and written observations.

## Notebooks

| # | Notebook | Dataset | Key techniques |
|---|----------|---------|----------------|
| 01 | `01_temperature_analysis.ipynb` | CDMX Monthly Temperatures 2023 | Axis logic, seasonal slicing, `np.diff`, Min-Max normalization |
| 02 | `02_sp500_returns.ipynb` | S&P 500 Monthly Returns 2023 | Arithmetic, `np.prod`, `np.cumprod`, rolling averages, `np.where` |
| 03 | `03_spotify_top50.ipynb` | Spotify Top 50 Mexico 2024 | `argsort`, multi-condition filters, `np.percentile`, weighted scoring |

## Data Sources

- **Temperature:** CONAGUA — https://smn.conagua.gob.mx
- **S&P 500:** Yahoo Finance — https://finance.yahoo.com/quote/%5EGSPC/history/
- **Spotify:** Spotify Charts — https://charts.spotify.com | BPM via https://tunebat.com
