# Pandas Practice Notebooks

Hands-on practice notebooks covering the core pandas library, built alongside
José Portilla's *Python for Data Science and Machine Learning Bootcamp* and
Jake VanderPlas's *Python Data Science Handbook*.

Each notebook uses a consistent video game sales dataset and focuses on
applied exploration rather than isolated exercises.

## Topics Covered

| Notebook | Topic |
|---|---|
| 01_series.ipynb | Series |
| 02_dataframes.ipynb | DataFrames |
| 03_conditional_filtering.ipynb | Conditional Filtering |
| 04_useful_methods.ipynb | Useful Methods |
| 05_missing_data.ipynb | Missing Data |
| 06_groupby_multiindex.ipynb | Groupby & MultiIndex |
| 07_combining_dataframes.ipynb | Combining DataFrames |
| 08_text_methods.ipynb | Text Methods |
| 09_time_methods.ipynb | Time Methods |
| 11_Pandas_project_exercise.ipynb | Final Project |

## Dataset

Video game sales data (real data sourced from Kaggle).
